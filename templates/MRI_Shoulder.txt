### Exam:
MRI Shoulder

### Technique:
MRI of the [right/left] shoulder was performed using PD-weighted fat-suppressed, T2-weighted, T1-weighted, and STIR sequences in coronal oblique, sagittal oblique, and axial planes. [If arthrogram: MR arthrography was performed following intra-articular gadolinium injection.]

### Clinical Details
**Instructions:**
- Record clinical details exactly as dictated. Do not rephrase or reformulate the wording.
- **Exclude** patient name, date of birth, age, gender, and MRN/accession numbers from the report, even if mentioned in the transcript.
- **If not provided:** 'No clinical details provided.'

### Comparison:
**Instructions:**
- **If provided:** List the prior imaging studies mentioned (modality and date/year if stated). Do not summarise changes here — comparison findings belong in the Findings section.
- **If not provided:** 'No prior imaging available for comparison.'

### Findings:
**Instructions:**
- Present findings using the bold section headers below in this exact order. Do NOT create new headings or merge sections.
- For each section: if the radiologist mentioned the structure (normal or abnormal), reproduce their exact words. If the radiologist did NOT mention it, use the default normal descriptor shown below.
- Within the Rotator Cuff section: when only one tendon is abnormal, describe it first, then state the remaining tendons (listed by name) as intact in a single line — do NOT create a sub-heading per tendon.
- Describe tear morphology precisely: partial vs full-thickness, location (critical zone / musculotendinous junction / footprint), and retraction distance if present.
- Normal structures within the same section may be combined into one statement. Only itemise individually when pathology is present.

**Rotator Cuff**
- Default if all intact: "Rotator cuff tendons (supraspinatus, infraspinatus, teres minor, subscapularis): intact, no tear."
- Default if one tendon abnormal: "[Describe abnormal tendon specifically.] Remaining rotator cuff tendons ([name the others]): intact."

**Long Head of Biceps Tendon**
- Default: "Long head of biceps tendon intact within the bicipital groove. No subluxation or dislocation."

**Labrum**
- Default: "Labrum intact. No labral tear."
- If pathology: report by compartment — anterior, posterior, superior (SLAP), inferior. Example: "Anterior labral tear at the 3 o'clock position without detachment. Posterior and superior labrum intact."

**Articular Cartilage**
- Default: "Glenohumeral articular cartilage intact. No focal chondral defect."

**Glenohumeral Joint**
- Default: "No significant glenohumeral joint effusion. Axillary recess unremarkable."

**Subacromial–Subdeltoid Bursa**
- Default: "No subacromial-subdeltoid bursitis or effusion."

**Acromioclavicular Joint**
- Default: "Acromioclavicular joint unremarkable. Coracoclavicular ligaments intact."
- If pathology: describe joint space narrowing, osteophytes, marrow oedema, or ligament disruption.

**Acromion**
- Default: "Type [I/II/III] acromion morphology. No os acromiale."

**Bony Structures**
- ALWAYS use the radiologist's exact dictated words for bony/marrow findings, whether normal or abnormal. Examples:
  - If they said "no fracture, no marrow oedema" → write exactly that.
  - If they said "Hill-Sachs lesion of the posterior humeral head" → write exactly that.
- Only use the default "Humeral head, glenoid, and clavicle: no fracture, subchondral cystic change, or marrow signal abnormality." if the radiologist said NOTHING about bony structures.
- If a specific abnormality is described, state it, then add "Remaining marrow signal is normal."

- If 'query' is mentioned in the transcript, it should be rendered as '?' in the report.

### Impression:
**Instructions:**
- Concise bullet-point summary of key findings.
- Include only clinically relevant findings. Minor incidental findings that are fully described in the body do not need to be restated here.
- For a normal/negative study, include a closing statement where appropriate, e.g. 'No cause for the patient's symptoms identified on this examination.'
- **Sample:**
  - Full-thickness supraspinatus tear at the critical zone with 1 cm retraction.
  - Labrum intact. No instability pattern.
  - Mild AC joint degeneration.
