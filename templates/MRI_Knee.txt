### Findings:
**Instructions:**
- Describe pathological structures specifically. Normal structures within the same anatomical group may be combined into a single statement — do not force a separate bullet for every structure when all are normal.
- Use precise radiology terminology for normal structures:
  - Menisci → "intact"
  - Ligaments → "intact"
  - Tendons → "intact"
  - Articular cartilage → "intact, no focal chondral defect"
  - Joint effusion → "no joint effusion"
  - Baker's cyst → "no Baker's cyst identified"
  - Bony structures → "no fracture or aggressive bony lesion"
  - Bone marrow → "no bone marrow oedema"
  - Soft tissues → "unremarkable"

Cover all groups in this order:

**Menisci**
- Medial Meniscus (anterior horn, body, posterior horn)
- Lateral Meniscus (anterior horn, body, posterior horn)

**Cruciate Ligaments**
- ACL and PCL (describe individually if pathology; otherwise a single combined statement is acceptable)

**Collateral Ligaments and Posterolateral Corner**
- MCL, LCL, and posterolateral corner structures (describe individually if pathology; otherwise combine normals)

**Articular Cartilage**
- If all surfaces are intact, a single statement suffices: "Articular cartilage intact throughout, no focal chondral defect."
- If pathology is present, report by compartment:
  - Medial Compartment (medial femoral condyle, medial tibial plateau)
  - Lateral Compartment (lateral femoral condyle, lateral tibial plateau)
  - Patellofemoral Compartment (patella, trochlear groove)
- Patellofemoral alignment and tracking are included here; do not create a separate Patellofemoral Joint heading.

**Tendons**
- Quadriceps Tendon and Patellar Tendon (combine if both normal)

**Soft Tissues**
- Joint Effusion, Hoffa's Fat Pad, Popliteal Fossa, Baker's Cyst

**Bony Structures**
- Baseline: "Bone marrow signal is normal. No suspicious osseous lesion identified."
- If an abnormality is described (e.g. bone marrow oedema, contusion, fracture), state that specifically, then add "Remaining marrow signal is normal."
